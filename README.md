# BrainfuckRvm
A Brainfuck interpreter with a JIT

- Based on https://eli.thegreenplace.net/2017/adventures-in-jit-compilation-part-2-an-x64-jit/

## Mandelbrot plot avg execution time

- VM1 = 45s
- VM2 = 30s
- VM3 = 10s
- JIT = 4s
- JITOpt = 2s
